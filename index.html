关于此仓库

这是一个非官方的 FFmpeg 预构建二进制文件仓库，提供方便快捷的 FFmpeg 最新版本下载。所有文件来源于官方 FFmpeg 构建，由于 GitHub 仓库限制上传超过 25MB 的文件，所有 FFmpeg 二进制文件已发布到 Releases 页面。

自动同步： 本页面会自动获取仓库中的最新版本，无需手动更新链接。

加速说明： 本页面已集成 gh-proxy.com 加速服务，如果您在中国大陆访问 GitHub 下载较慢，可以使用加速下载链接。

安装与使用说明

Windows 系统

  1. 下载 ffmpeg-*-essentials_build.zip
  2. 解压到任意目录（例如 C:\ffmpeg）
  3. 将 bin 目录添加到系统 PATH 环境变量
  4. 或在命令行中直接使用完整路径运行

Linux 系统

Linux 安装命令 
# 下载并解压
wget [下载链接]
tar -xf ffmpeg-release-amd64-static.tar.xz
cd ffmpeg-[版本号]-amd64-static/

# 运行 FFmpeg
./ffmpeg

# 安装到系统路径
sudo cp ffmpeg ffprobe /usr/local/bin/

验证安装

验证命令 
# 检查版本
ffmpeg -version

# 查看支持的编解码器
ffmpeg -codecs

# 测试基本功能
ffmpeg -i input.mp4 output.avi

常见问题

如何设置 PATH 环境变量？

Windows: 右键点击"此电脑" → 属性 → 高级系统设置 → 环境变量 → 在系统变量中找到Path → 编辑 → 添加FFmpeg的bin目录路径

Linux/macOS: 在 ~/.bashrc 或 ~/.zshrc 中添加: export PATH=$PATH:/path/to/ffmpeg/bin

下载速度很慢怎么办？

请使用"加速下载"按钮，该链接通过 gh-proxy.com 代理，可以显著提高在中国大陆的下载速度。
